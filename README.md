# JClip

A sort of clipboard manager that appear below your cursor.

## Getting Started
```console
javac Clipboard.java
java Clipboard &
```
## KeyBinding

|Key|Definition|
|---|---|
|`k`|Move Up|
|`j`|Move Down|
|`c`|Change Selected Item|
|`d`|Delete Selected Item|
|`i`|Add a new Item|
